# DAC Implementation Projects (MPLAB)

This folder contains multiple Digital-to-Analog Conversion (DAC) implementations using Embedded C in MPLAB. The projects focus on generating different analog outputs using microcontroller-based DAC techniques.

## Projects Included

### 1. DAC Basic (adcdac)
- Demonstrates basic DAC functionality
- Converts digital values into corresponding analog output
- Useful for understanding DAC initialization and output mapping

### 2. DAC 2.5V Output (DAC2.5V)
- Generates a fixed analog output of approximately 2.5V
- Demonstrates precise voltage control using DAC
- Useful for calibration and reference voltage applications

### 3. DAC Square Wave (DACSquare)
- Generates a square wave using DAC
- Demonstrates waveform generation using digital control
- Can be extended to signal generation applications
