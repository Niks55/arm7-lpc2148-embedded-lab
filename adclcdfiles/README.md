# ADC Interfacing using MPLAB

This project demonstrates Analog-to-Digital Conversion using microcontroller programming in C.

## Features
- ADC initialization and configuration
- Reading analog input
- Displaying digital output

## Files
- `adc.c` → ADC logic
- `adc.h` → header definitions
- `main.c` → main program
- `delay.c/h` → timing functions

## Concepts Used
- ADC conversion
- Embedded C programming
- Hardware interfacing
